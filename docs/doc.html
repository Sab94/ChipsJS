Installation

Basic installation

Chips relies on jQuery (>= 2), and it will need to be included before including Chips.

Then, you can include chips.css and then chips.min.js.

That would look pretty much like this:

            <script src="jquery.js"></script>
<link rel="stylesheet" type="text/css" media="screen" href="chips.css" />
<script src="chips.js"></script>

Javascript installation

Like for Basic installation, first include jQuery and Chips. Then, simply use $('input').chips(options); or new Chips('input', options); (where options is an optional configuration object).

That would look pretty much like this:

          <script src="jquery.js"></script>
<link rel="stylesheet" type="text/css" media="screen" href="chips.css" />
<script src="chips.js"></script>


<input id="myChipInput">


<script>
  $('#myChipInput').chip();
</script>

Usage

Initialization

$ API Return
$('#existingInput').chip(options) chipInstance
$('#notExistingDOMElement').chip(options) undefined

Options

Property Description
separetor Set an array of keys to trigger the creation of a chip
separetor : [' ', ',', '+', '-', 'enter']
**available list od separators are listed below.
chipStyle Assign style attribute to the chips.
eg: chipStyle : "color:white;background:#F44336;".
wrapperStyle Assign style attributes to the input wrapper: wrapperStyle: 'width: 80%;margin: 10px;'
type Set type of the input field. Default is text.
type: 'email', will check if the input has a valid email.
**for now only text and email is supported. others coming soon...
activeOnBlur Create chip from input on blur. Default is false.
activeOnBlur: true, will create chip from input text on blur event.
md Material design chips Default is false.
md: true, will create set the chip style to material design.
target Create chips inside some other html div element, eg: target: '#target'.
By default it will be a container inside the wrapper.
containerStyle Assign style attribute to the container. containerStyle: 'display: flex;flex-wrap: wrap;justify-content: center;'.

Methods

Method Returns Description
load([]) Load chips from an array of data.
get() String get comma seperated string of chip texts.
isValid() boolean Returns a boolean. true if atleast one chip exist. false otherwise.
reset() Reset Chips for this field.
destroy() Disable and destroy Chips for this field.
